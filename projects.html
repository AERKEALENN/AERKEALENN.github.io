QS魔方计时

多人魔方计时器，支持键盘/点击记录成绩、自动排名、金银铜高亮及时间差显示。

Tauri 2.0 React 19 TypeScript
功能特性
  • 极光粒子动态背景
  • 自定义无边框标题栏（拖拽/最小化/最大化/关闭）
  • 键盘任意键记录选手成绩
  • 点击圆环区域也可触发
  • 成绩榜自动排名，金银铜高亮
  • 时间差（gap）显示
技术栈
  • Tauri 2.0 + Rust 后端
  • React 19 + TypeScript
  • Vite 构建
